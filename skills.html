Effect of Poisson Distribution on Imaging

• This study is done to see the effect of Poisson distribution on signals. Here, I took specific number of Poisson distribution of each single pixel of a grayscale image with Poisson parameter lambda equal to the pixel brightness of the pixel in 8-digit scale.
• As number of consecutive Poisson distributions increases, the noise in the image reduces. The graphs on bottom right supports this claim since these graphs indicate the absolute mean error between original image and decayed image for every single number of Poisson distributions.
• Another way to see the difference between these images quantitatively, we can analyze the difference between pixel brightness graphs on bottom left. The peaks in "Observation Matrix" indicate the noise generated.
• The reason the effect of Poisson distribution analyzed here is, this probabilistic fact is going to be used in further studies on predicting the Source-Shot Noise, which is a challenge for imaging techniques using electron beam microscopy.
